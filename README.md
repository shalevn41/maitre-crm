# 🍽️ MAITRE — Restaurant CRM & Marketing Automation

> ### 🚧 Work in Progress — This project is actively being developed and is not yet final.

A full SaaS platform for restaurant loyalty, marketing automation,
and customer relationship management — built with Base44.

## ✨ Features

- 👥 Customer loyalty club with points & rewards
- ⭐ Automated Google Reviews management
- 📱 WhatsApp marketing automation
- 💳 Credits-based monetization system
- 👨‍💼 God Mode super-admin dashboard
- 🍽️ Waiter incentives system via QR codes
- 📊 CRM with full customer history
- 📣 Campaign management & templates

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| Base44 | Full-stack no-code builder |
| React + Vite | Frontend framework |
| Tailwind CSS | Styling |
| Supabase | Database & auth |
| WhatsApp API | Marketing messages |
| Google Reviews API | Review automation |
| Stripe | Payments |

## 🚀 How It Works

1. Restaurant signs up and registers via QR code
2. Customers join loyalty club through waiters
3. System tracks visits, points, and rewards automatically
4. Manager sends WhatsApp campaigns to segmented customers
5. Google Reviews requests sent automatically after visits
6. God Mode admin monitors all restaurants in real-time

## 💰 Business Model

- Single pricing tier: ₪499/month per restaurant
- Waiters act as acquisition agents via QR codes
- Moat: network effect through waiter incentives
